我是如何利用 iPad 来学习的？

发表于 2020-02-25 更新于 2020-03-10 分类于 经验分享 阅读次数：
本文字数： 阅读时长 ≈ NaN:aN

2017 年，也就是进入大学的第一年，入手了 32G 的 iPad 2017 以及 Apple Pencil，到现在已经有两年半了。我主要用 iPad 来学习（当然偶尔也会玩玩音游），基本上我的所有笔记都是用 iPad 记的，以及大部分的书和文献都是用 iPad 来阅读整理的。对我来说，iPad 是我至今买过的最值的东西了，没有之一（顺便透露一下，第二值的是 Wallpaper Engine）。由于我在高中时基本上不记笔记，也没有接触过 iOS 系统，所以对苹果的生态的了解是从零开始的，到现在差不多摸索出了一套比较合适的无纸化学习的工作流。

日常使用场景以及手写笔记方法

下面我分不同的使用场景来分享我记笔记的经验。配上 Pencil 的 iPad 真的是非常适合手写笔记，所以我就主要讲手写笔记了。

课堂笔记

我的第一个使用场景是用 iPad 做课堂笔记，用到的软件是 Notability。

现在大部分的大学老师都是用 PPT 来上课，而且一般上课的 PPT 老师都会提供给学生。但是如果因为这样就不做课堂笔记是不太明智的。不过另一方面，也因为 PPT 能够拿到手，所以课堂笔记没有必要记得很详细，更多的时候，需要记的是一些事后容易忘记的地方和一节课整体的逻辑脉络。拿我的专业，物理，来说，一般一节专业课讲的内容非常多，而且常常会出现大面积的公式推导。那么我的做法是把课上出现的概念用思维导图的形式记下来，然后公式推导的话，我只会记下自己不容易想到的步骤。下面是我大二学习数学物理方法的时候做的一个复变函数课堂笔记。

课堂笔记的这个特点要求记录的速度要快，而且要便于课后回顾的时候能想起来这节课大概在干什么。这里我选择的软件是 Notability。Notability 是我用过的笔记软件里我觉得最适合用来做课堂笔记的，或者说最适合做快速记录的。Notability 的特点是淡化了传统意义上笔记本的概念和页的概念，在 Notability 上创建的一份笔记（软件官方称为“备注”）就像一张无限长的纸一样，一边写可以一边往下拉，这样的好处是在记笔记的时候不用考虑“这一页马上要写完了，但是要记的内容还没完全写下来，要不要换到下一页来写？”“这一页的空间好像不够了，我是不是应该调整一下上面的排版？”之类的问题。这一点 Notability 和隔壁的 Goodnotes （iOS 段另一款笔记软件） 不一样，在 Goodnotes 里传统的笔记本的概念被保留着，笔记是一本一本的，这样，Goodnotes 更适合用来做后期的整理笔记，或者做习题册和错题集。

整理笔记

另一个使用场景是在课后自学和复习时整理笔记，用到的软件也是 Notability。用 Goodnotes 也行，主要看个人的爱好，我更偏好 Notability。实际上，在之前一段时间这两个软件的大更新后，两者书写体验的区别已经很小了。

顾名思义，整理笔记的作用就是对学习的内容进行整理。有一个费曼学习法

费曼（20 世纪物理学家）曾经说，检验自己是否弄懂了的最好办法就是向外行人解释这个内容，如果能让他搞懂，就说明自己确实弄懂了这个内容。

因此，学习的最好的办法就是多和同学讨论。但是呢，也不是所有时候都有人来和你讨论，这个时候整理笔记就可以起到类似的作用。

整理笔记要怎样写？首先，我并不推荐为了以方便日后复习为目的来写整理笔记，页更不推荐把大量的心思放在怎样让笔记变得好看上。这些东西都是次要的，是锦上添花的东西。整理笔记呢，就像上面说的，目的是为了整理思路。在写整理笔记的时候，最好抱着“这个笔记要让外行人也能看懂”的心态来写。

对于像物理学（绝大部分的理科也是）这种逻辑性很强的学科，写整理笔记的过程，就是在整理逻辑，要让每一处的逻辑都能连接上。我一般有两种做笔记的习惯，一个是跟着书的顺序，边看边记，另一个是在看完书的一章或一部分后，关上书自己按照合适的逻辑来记笔记，当然必要的时候（比如忘了某个概念的确切定义的时候）还是要翻翻书的。这两种方法适合不同的情形，像理论性比较强的科目，如物理系的四大力学（理论力学，量子力学，电动力学，统计力学），就适合在看完书后自己写笔记；另一些知识点比较多的，理论性不那么强的，比如模拟电子电路、光学、原子物理等，就适合边看边记了。

文献阅读

iPad 的另一大作用就是用来读文献了。为了找一款合适的 PDF 阅读软件，我也走了很多坑。最后感觉最适合学习的是 Marginnote 3。

Marginnote 3 是专门为学习设计的一款 PDF 阅读软件。与其他只能打开 PDF 的阅读器不同，Marginnote 为每一个 PDF 都准备了一个脑图笔记本，你可以在阅读的同时把你觉得重要的地方拖到脑图笔记本里，然后整理成思维导图。另外，Marginnote 3 还有一个最厉害的功能，就是 OCR，这个功能可以自动识别 PDF 里的文字，再也不用因为扫描版的 PDF 不能复制文字发愁了，我也是因为这个功能而把 Marginnote 3 作为我的主力文献阅读软件的。不管是教材还是论文，我都是用 Marginnote 3 来看的。下面是我为这学期在学的生物物理导论的绪论部分做的思维导图，当然，还没有做完。

除了 Marginnote 3 之外，我还用过 PDF Expert 和 LiquidText，前一个比较常规，后一个很有特色，有兴趣的可以去了解。

颜色的妙用

这里分享一下我在阅读文献的时候，使用颜色标记的方法，这套方法我也在不断完善

  • 红色：非常重要和值得重视的内容
    • 重要的事实、定律、定理等
    • 容易混淆的东西之间的对比
    • 与常识违背的事实
    • 容易搞错的地方
  • 绿色：构成文章整体脉络的内容
    • 没有达到被标红的等级的事实、定律、定理
    • 概念的定义
  • 黄色：补充内容
    • 概念的解释
    • 重要的论证
    • 补充的图形、表格等
  • 粉红色：有待日后解决的内容
    • 没看懂的地方
    • 存在疑问的地方
    • 需要查资料补充的地方

PS：关于我为什么选择了 32G 的 iPad？

当时以为如果只用来学习的话，32G 应该绰绰有余了。而且因为我的上一个平板储存空间才 16G，我天真地以为 32G 已经够大了。实际的情况是，在一开始的一年里，32G 确实够了。当时我的 iPad 上比较常用的软件有 Notability、Marginnote、Texpad、Microsoft Office 全套还有音游 Cytus 2，这个时候我的 iPad 储存空间一般会用掉 18G 左右。但是，随着后来 iPad 里存的文档变多（像我的专业，物理，用到的电子书，如果是扫描版的话，一般一个 PDF 就会占掉 100Mb），空间变得紧张起来了。空间变小的一个后果就是，如果你在设置里允许你的 iPad 当自动删除一些文件来释放储存空间的话，就有可能出现在你急着要用某个文档的时候你发现它被删掉了，还得重新从 iCloud 上下回来。甚至我还遇到过正在用 Notability 写的笔记被删掉的情况。后来我不得不卸掉了 Cytus 2。

所以如果你有入手一台 iPad 的打算，那我的建议是，如果不是真的缺钱，那就不要吝啬那么几百块了，储存空间宁大勿小，毕竟对于普通的学生党来说，iPad 的更换周期是比较长的，那为什么不一开始就把储存空间买大一点呢？
