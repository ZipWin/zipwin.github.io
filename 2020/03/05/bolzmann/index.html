玻尔兹曼分布的计算机模拟

发表于 2020-03-05 更新于 2020-03-10 分类于 统计力学 阅读次数：
本文字数： 4.5k 阅读时长 ≈ 4 分钟

正则系综的系统，能量的概率分布满足玻尔兹曼分布

计算机模拟可以看到正则系综建立玻尔兹曼分布的过程. 假设现在来做这样一个实验：有大量可区分的粒子，一开始每个粒子都有 $1$ 份能量，之后随机拿掉某个粒子的 $1$ 份能量，将这份能量随机放到另一个粒子上，反复重复这样的过程。计算机模拟发现，重复足够多次后，可以发现，具有一定能量的粒子总数关于能量的分布是服从玻尔兹曼分布的.

模拟一：无空间结构的粒子体系

Problem Setup

  1. 把 $N^2$ 个粒子摆成 $N\times N$ 的矩阵
  2. 每个粒子初始带有 $1$ 份能量
  3. 不断重复
    1. 随机选取一个能量不为零粒子，称为给与者
    2. 随机选取一个给予者之外粒子，称为接收者
    3. 给与者能量数减 $1$
    4. 接收者能量数加 $1$
  4. 统计能量数为 $E$ 的粒子个数 $N$
  5. 验证 $N$ 关于 $E$ 的分布是否符合玻尔兹曼分布

这个 Setup 中，给予者和接收者是随机选取的两个不同粒子，虽然这 $N^2$ 个粒子被排成了 $N\times N$ 的矩阵，但是这个结构并没有在操作过程中体现，因此把这个模拟叫做无空间结构体系的模拟. 实际上，改变粒子的排列并不会影响结果，这里排成矩阵是为了方便拓展到有空间结构的模拟（下面会看到，也就是只允许能量在相邻的粒子间转移）.

模拟二：有空间结构的粒子体系（无界空间）

Problem Setup

  1. 把 $N^2$ 个粒子摆成 $N\times N$ 的矩阵
  2. 每个粒子初始带有 $1$ 份能量
  3. 不断重复
    1. 随机选取一个能量不为零粒子，称为给与者
    2. 随机选取一个与给予者相邻的粒子（有8个）（越过边界也认为是相邻的），称为接收者
    3. 给与者能量数减 $1$
    4. 接收者能量数加 $1$
  4. 统计能量数为 $E$ 的粒子个数 $N$
  5. 验证 $N$ 关于 $E$ 的分布是否符合玻尔兹曼分布

与模拟一不同的是，这里只允许粒子与它周围的粒子交换能量，并在四个边界上添加了周期性边界条件.

模拟三：有空间结构的粒子体系（有界空间）

Problem Setup

  1. 把 $N^2$ 个粒子摆成 $N\times N$ 的矩阵
  2. 每个粒子初始带有 $1$ 份能量
  3. 不断重复
    1. 随机选取一个能量不为零粒子，称为给与者
    2. 随机选取一个与给予者相邻的粒子（除边界的粒子外有8个）（不能越过边界），称为接收者
    3. 给与者能量数减 $1$
    4. 接收者能量数加 $1$
  4. 统计能量数为 $E$ 的粒子个数 $N$
  5. 验证 $N$ 关于 $E$ 的分布是否符合玻尔兹曼分布

与模拟二一样只允许周围的粒子交换能量，只是这里边界是封闭的，粒子不能跨过边界交换能量.
